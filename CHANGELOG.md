# Changelog

All notable changes to this project are documented in this file. The format is
based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this
project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-07-13

### Added

- Codecs for four HTTP-fixture dialects, converting through one shared
  exchange model: HAR 1.2, VCR cassettes in both the Python vcrpy
  (`interactions`) and Ruby VCR (`http_interactions`) dialects, WireMock
  stub mappings (single file and `mappings` export), and nock definitions
  (`nock.recorder` / `nock.define`).
- `yamlite`, a dependency-free YAML subset reader/writer covering exactly
  what psych and PyYAML emit for cassettes: block collections, compact
  sequence entries, multi-line folded plain/quoted scalars, block scalars
  with chomping, flow collections, comments, and `!!binary`; anchors,
  custom tags, and multi-document streams are rejected with clear errors.
- Structural format detection from content shape alone — no reliance on
  file extensions, refusal (`DetectError`) instead of guessing.
- Lossless carry of methods, full URLs, ordered multi-valued headers, text
  and binary bodies (base64/hex/`!!binary` per dialect), status codes, and
  recording timestamps (ISO 8601 ⇄ RFC 2822) wherever the target dialect
  has a field for them; the full support map lives in
  `docs/format-matrix.md`.
- Explicit degradation everywhere else: warning notes on stderr by default;
  when the *target* cannot hold the data, `--strict` raises
  `UnsupportedFeatureError` instead (WireMock's hostless stubs, nock binary
  request bodies, vcrpy-JSON binary bodies).
- Semantic equivalence checking (`compare_fixtures`) with RFC 7230 header
  grouping, JSON-aware body comparison, and URL normalization.
- `fixmux` CLI: `convert` (auto-detect, `--strict`, `--base-url`,
  `--vcr-serializer yaml|json`), `detect`, `inspect`, `verify` (exit 1 on
  drift), and `formats`; reads stdin via `-`, keeps stdout pipeable.
- Deterministic output: byte-identical conversions run-to-run, sorted keys
  matching the native recorders, fixed epoch sentinel for missing
  timestamps.
- Runnable examples: the same two exchanges expressed in all four dialects,
  generated by fixmux itself (`examples/`).
- 92 pytest tests (including a 20-path source×target round-trip matrix) and
  `scripts/smoke.sh`, an end-to-end CLI smoke run printing `SMOKE OK`.

### Notes

- The repository ships no CI workflow; verification is local —
  `pip install -e '.[dev]' && pytest && bash scripts/smoke.sh`.
